# /explain-code

Explain the code in detail:

1. **Overview**: What does this code accomplish?
2. **Step-by-Step**: Walk through the logic
3. **Key Concepts**: Explain patterns and techniques used
4. **Dependencies**: What does it rely on?

Explanation Levels:
- High-level purpose
- Function-by-function breakdown
- Line-by-line for complex sections
- Data flow explanation

Include:
- Plain language explanations
- Technical details where relevant
- Analogies for complex concepts
- Visual diagrams if helpful (ASCII)

Adjust explanation depth based on code complexity.
