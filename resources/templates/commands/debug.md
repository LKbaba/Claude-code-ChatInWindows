# /debug

Help debug the issue:

1. **Analyze the Problem**: Understand the symptoms
2. **Trace the Code**: Follow the execution path
3. **Identify Causes**: Find potential root causes
4. **Suggest Solutions**: Provide debugging steps

Debugging Approach:
- Check error messages and stack traces
- Verify input data and types
- Check for null/undefined values
- Verify async/await handling
- Check state management
- Review recent changes

Provide:
- Likely cause of the issue
- Debugging steps to confirm
- Suggested fix
- Prevention strategies
