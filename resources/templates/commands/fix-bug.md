# /fix-bug

Analyze the bug and provide a fix:

1. **Identify the Issue**: Explain what's causing the bug
2. **Root Cause Analysis**: Why does this bug occur?
3. **Proposed Fix**: Provide the corrected code
4. **Prevention**: How to prevent similar bugs in the future

Steps:
1. Reproduce the issue mentally by tracing the code
2. Identify the exact location of the bug
3. Understand the expected vs actual behavior
4. Implement the minimal fix that resolves the issue
5. Consider edge cases the fix might affect

Provide the fixed code with clear comments explaining the changes.
